Telephone Directory — Hashing
Built a high-performance telephone directory application as part of a Data Structures course, tackling one of the most fundamental challenges in computer science — fast and efficient data lookup at scale. The system was engineered using custom hash table implementations with both linear and quadratic probing strategies to handle collision resolution effectively. This project deepened my understanding of time-space tradeoffs, memory efficiency, and the real-world impact of algorithmic choices on application performance.

05 — Research

Published Paper

IRJET · Vol. 8, Issue 1 · Jan 2021
KLEF Play — A Progressive Web App for Institutional Video Streaming
Co-authored research published in the International Research Journal of Engineering and Technology (IRJET) (Impact Factor: 7.529). KLEF Play is a video streaming Progressive Web App that leverages YouTube as a storage and transcoding backend while keeping full content control within the institution. Built with React.js, Node.js, and MongoDB, the PWA architecture ensures cross-device compatibility, offline support, and significantly reduced server overhead — saving over 90% of server storage compared to self-hosted solutions.
